Promogoose

January 2019 - March 2019

www.promogoose.fr

English

Promogoose is a teamwork school project I made during my Web development's Professional degree. The primary motivation for the user is to save / earn money but it is also about offering a quality service. A user who comes to the site must be able to say that he has the privilege of bargains.

  1. Allow merchants to manage and offer flash promotions.
  2. Allow the user to quickly visualize on a map stores or structures offering discounts according to its geographical position.
  3. Allow the user to search via keywords or categories, places etc...
  4. Allow the user to put notices on a promotion or businesses.
