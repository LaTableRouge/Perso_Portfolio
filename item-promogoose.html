English

Promogoose is a teamwork school project I made during my Web development's Professional degree. The primary motivation for the user is to save / earn money but it is also about offering a quality service. A user who comes to the site must be able to say that he has the privilege of bargains.

  1. Allow merchants to manage and offer flash promotions.
  2. Allow the user to quickly visualize on a map stores or structures offering discounts according to its geographical position.
  3. Allow the user to search via keywords or categories, places etc...
  4. Allow the user to put notices on a promotion or businesses.
Tools
  • Laravel
  • Photoshop (for design)
  • PHP, SQL, HTML, SASS, JS
  • Leaflet's library
  • GIT (for teamwork)

French

Promogoose est un projet tuteuré fait en équipe lors de ma licence professionnelle Mobilité Internet Web. La motivation première pour l'utilisateur de cette application web est de faire des économies d'argent mais c'est aussi un service de qualité. Un utilisateur qui utilise l'application web, doit pouvoir se dire qu'il possède le privilège de la promotion.

  1. Permettre aux commerces de gérer et d'offrir des promotions flash
  2. Permettre à l'utilisateur de visualiser sur une carte les magasins et structures qui offrent des promotions selon sa position géographique
  3. Permettre à l'utilisateur de rechercher via mots-clés, catégories, endroits etc...
  4. Permettre à l'utilisateur de mettre des notes et de laisser un commentaire sur une promotion ou un magasin
Outils
  • Laravel
  • Photoshop (pour le design)
  • PHP, SQL, HTML, SASS, JS
  • La bibliothèque Leaflet
  • GIT (pour le travail en équipe)
