www.promogoose.fr January 2019 - March 2019 2

Supervised project

English
Promogoose is a teamwork school project I made during my Web development's Professional degree. The primary motivation for the user is to save / earn money but it is also about offering a quality service. A user who comes to the site must be able to say that he has the privilege of bargains.
Tools
  • Laravel
  • Photoshop (for design)
  • PHP, SQL, HTML, SASS, JS
  • Leaflet's library
  • GIT (for teamwork)
Français
Promogoose est un projet tuteuré fait en équipe lors de ma licence professionnelle Mobilité Internet Web. La motivation première pour l'utilisateur de cette application web est de faire des économies d'argent mais c'est aussi un service de qualité. Un utilisateur qui utilise l'application web, doit pouvoir se dire qu'il possède le privilège de la promotion.
Outils
  • Laravel
  • Photoshop (pour le design)
  • PHP, SQL, HTML, SASS, JS
  • La bibliothèque Leaflet
  • GIT (pour le travail en équipe)
